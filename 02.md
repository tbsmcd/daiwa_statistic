# 母集団と標本分布  
## 分布
### 正規分布  
- σ 標準偏差（散らばり具合)
- μ 平均値  
- σ = Σ(Xn-μ)
- 正規分布だと σ = 0 （左右対称の分布だから）なので、（正規分布全体の）分散 = Σ(Xn-μ)^2 を使う
- 分散/n = 測定値1個あたりの分散  
- σ = √分散
  
正規分布の横軸は測定値 x 縦軸は確率密度 f(x)

μ +- σ の合計: 68%
μ +- 2σ の合計: 95.45%  
  
- 全空間の確率密度の積分は1
- 全空間のxf(x) = 測定値x確率密度の積分は期待値（=μ）
  
### 標準正規分布
- μ = 0
- σ = 1
- σ^2 = 1
