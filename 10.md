# 直線回帰
## 考え方
### 例
- Y = a + bx
	- 回帰曲線
	- (実験条件,実測値) = (x1,y1), (x2,y2), ...(xn,yn)  から得られるものとする
	- a, b は回帰係数
- 回帰残差の平方和
	- `S = Σ[i→n](yi - Yi)^2 = Σ[i→n](yi - a - bxi)^2`
	- （↑`Y = a + bx` より）
	- グラフ上の点 Yi と 実測値 yi のズレ
	- S が最小になる場合が最適な直線
		- 条件
			- 二次関数 S = f(a) なので `dS/da = 0` になる a
			- 二次関数 S = f(b) なので `dS/db = 0` になる b
